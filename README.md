### Основные направления работы

- **Backend & Python:** Разработка микросервисов, проектирование баз данных, написание масштабируемого кода.
- **Telegram-боты:** Архитектура любой сложности (машины состояний FSM), платежные шлюзы, интеграция со сторонними API и парсинг.
- **Интеграции с 1С:** Разработка внешних модулей, обработок и API для надежного обмена данными между базами 1С и веб-сервисами.
- **Инфраструктура:** Упаковка проектов в Docker, настройка CI/CD пайплайнов, деплой на Linux-серверы.
